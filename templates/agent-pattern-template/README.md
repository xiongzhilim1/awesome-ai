# Agent Pattern Template

A minimal, extensible agent pattern demonstrating:
- tool calling
- memory
- planning loops
- error handling
- observability points

Use this as the base for more complex agent systems.

## Files
- `agent.py` — main agent logic
- `tools.py` — tool functions + schemas
- `memory.py` — short-term + long-term memory
- `planner.py` — decision-making logic
